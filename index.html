https://sjrtd.azurewebsites.net

Routes

Endpoint Description
/routes returns all routes
/routes?type=[RouteType] Returns all routes of matching type. Replace [RouteType] with Local Hopper-Local Hopper-County BRT-Express Commuter. Case Sensitive on Route Type.
/routes/paths Returns all paths for all routes.
/routes/paths?type=[RouteType] Returns all paths of matching type. Replace [RouteType] with Local Hopper-Local Hopper-County BRT-Express Commuter. Case Sensitive on Route Type.

Fares

Endpoint Description
/fares returns all fares
/fares?type=[RouteType] Returns all fares of matching type. Replace [RouteType] with Local Hopper-Local Hopper-County BRT-Express Commuter. Case Sensitive on Route Type.

Auth

Endpoint Description
/auth/add Adds user to database
login If login JSON data matches a user in database, will return JwtSecurityToken
